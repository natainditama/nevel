<!-- Prerequisites -->
### ğŸ”§ Prerequisites

Prerequisites for this project:

- [Nodejs](https://nodejs.org)
- [Yarn](https://yarnpkg.com/)

<!-- Run Locally -->
### ğŸƒ Run Locally

Clone the project

```bash
  git clone https://github.com/natainditama/nevel.git
```

Go to the project directory

```bash
  cd nevel
```

Install dependencies

```bash
  yarn install
```

Start the local server

```bash
  yarn run dev
```
